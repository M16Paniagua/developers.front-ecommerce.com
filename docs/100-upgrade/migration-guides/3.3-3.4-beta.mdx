<p>{frontMatter.description}</p>

:::caution

The 3.4.0 Version is still in beta stage, we don't recommend you to update your production code to this release.

:::

## Update dependencies

Update all your `@front-commerce/*` dependencies to this version:

```shell
pnpm update "@front-commerce/\*@3.4.0-beta.0"
```
